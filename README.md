# Versions used:

- _node v5.6.0_

- _truffle v4.1.14_

- _web3 v1.0.0-beta.4_


### Installation

Install the dependencies

```sh
$ cd project-6
$ npm install
```
Launch Ganache:

```
$ ganache-cli -m "your mnemonic goes here"
```
Compile smart contracts

```sh
$ truffle compile
```

Migrate smart contracts to the locally running blockchain, ganache-cli:

```sh
$ truffle migrate
```

Test smart contracts:

```sh
$ truffle test
```

Launch the DApp:
```
$ npm run dev
```
